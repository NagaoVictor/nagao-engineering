\# Comparações



As comparações permitem estabelecer relações entre valores.



| Operação | Símbolo |

|----------|---------|

| Menor | \&lt; |

| Maior | \&gt; |

| Menor ou igual | \&le; |

| Maior ou igual | \&ge; |

| Diferente | \&ne; |

\# Matemática



Inline:



```text

a + b

```



Exemplo de expressão:



```text

a + b = c

```
