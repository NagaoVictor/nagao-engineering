\# Comparações



As comparações permitem estabelecer relações entre valores.



| Operação | Símbolo | Exemplo | Leitura |

|----------|----------|----------|----------|

| Igual | $=$ | $a=b$ | a é igual a b |

| Diferente | $\\neq$ | $a\\neq b$ | a é diferente de b |

| Menor | $<$ | $a<b$ | a é menor que b |

| Maior | $>$ | $a>b$ | a é maior que b |

| Menor ou igual | $\\leq$ | $a\\leq b$ | a é menor ou igual a b |

| Maior ou igual | $\\geq$ | $a\\geq b$ | a é maior ou igual a b |

\# Operações Fundamentais



As operações fundamentais são utilizadas para transformar valores numéricos.



| Operação | Símbolo | Exemplo |

|----------|----------|----------|

| Soma | $+$ | $a+b$ |

| Subtração | $-$ | $a-b$ |

| Multiplicação | $\\times$ | $a\\times b$ |

| Divisão | $\\frac{a}{b}$ | $\\frac{a}{b}$ |

| Potência | $^n$ | $a^n$ |

| Raiz | $\\sqrt{x}$ | $\\sqrt{x}$ |

| Valor absoluto | $|x|$ | $|x|$ |

| Módulo | $\\bmod$ | $a\\bmod b$ |

\# Conjuntos



Os conjuntos representam coleções de elementos.



| Conceito | Símbolo | Exemplo |

|----------|----------|----------|

| Pertence | $\\in$ | $x\\in A$ |

| Não pertence | $\\notin$ | $x\\notin A$ |

| Subconjunto | $\\subseteq$ | $A\\subseteq B$ |

| Subconjunto próprio | $\\subset$ | $A\\subset B$ |

| União | $\\cup$ | $A\\cup B$ |

| Interseção | $\\cap$ | $A\\cap B$ |

| Diferença | $A-B$ | $A-B$ |

| Produto Cartesiano | $\\times$ | $A\\times B$ |

| Cardinalidade | $|A|$ | $|A|$ |

| Conjunto vazio | $\\varnothing$ | $\\varnothing$ |



\---



\# Quantificadores



Os quantificadores descrevem condições gerais sobre elementos.



| Conceito | Símbolo | Leitura |

|----------|----------|----------|

| Para todo | $\\forall$ | Para todo |

| Existe | $\\exists$ | Existe |

| Existe único | $\\exists !$ | Existe um único |



\---



\# Lógica



Os operadores lógicos são utilizados para construir proposições.



| Operação | Símbolo |

|----------|----------|

| Negação | $\\neg P$ |

| Conjunção | $P\\land Q$ |

| Disjunção | $P\\lor Q$ |

| XOR | $P\\oplus Q$ |

| Implicação | $P\\Rightarrow Q$ |

| Equivalência | $P\\Leftrightarrow Q$ |



\---



\# Funções



Uma função associa elementos de um conjunto a elementos de outro conjunto.



| Conceito | Sintaxe |

|----------|----------|

| Função | $f:A\\rightarrow B$ |

| Aplicação | $f(x)$ |

| Domínio | $Dom(f)$ |

| Contradomínio | $Cod(f)$ |

| Imagem | $Im(f)$ |
