\# Provas de Existência



\## O que é?



Uma prova de existência demonstra que \*\*pelo menos um objeto\*\* com determinada propriedade realmente existe.



Diferente das provas construtivas, uma prova de existência \*\*nem sempre precisa mostrar qual é esse objeto\*\*. Em alguns casos, basta provar que ele obrigatoriamente existe.



As provas de existência podem ser:



\- \*\*Construtivas\*\* → mostram o objeto.

\- \*\*Não construtivas\*\* → apenas demonstram que ele existe.



\---



\## Exemplo 1 — Prova Construtiva



Prove que:



> \*\*Existe um número par maior que 50.\*\*



\---



\## Como resolver



Basta apresentar um exemplo.



```text

52

```



Verificando:



```text

52 > 50



52 ÷ 2 = 26

```



Como o número satisfaz todas as condições, concluímos que a afirmação é verdadeira.



✔ A existência foi demonstrada apresentando o próprio objeto.



\---



\## Exemplo 2 — Prova Não Construtiva



Imagine a afirmação:



> \*\*Existe um número natural que satisfaz determinada propriedade.\*\*



Suponha que, por algum teorema já conhecido, seja possível provar que esse número obrigatoriamente existe, mas que encontrá-lo seja muito difícil ou até impossível na prática.



Nesse caso, podemos concluir:



```text

O objeto existe.

```



Mesmo sem saber exatamente qual é ele.



Essa é uma prova de existência não construtiva.



\---



\## O que aprender com isso



O objetivo de uma prova de existência não é responder:



> \*\*"Qual é o objeto?"\*\*



Mas sim:



> \*\*"Ele realmente existe?"\*\*



Em muitos problemas da matemática, provar a existência já é um grande avanço, mesmo sem encontrar um exemplo específico.

\## Resumo



| Conceito | Ideia |

|----------|-------|

| \*\*Prova de existência\*\* | Demonstra que pelo menos um objeto com certa propriedade existe. |

| \*\*Construtiva\*\* | Mostra exatamente qual é o objeto. |

| \*\*Não construtiva\*\* | Prova a existência sem identificar o objeto. |



\---



\## Modelo mental



Imagine que alguém diga:



> "Existe um caminho para sair deste labirinto."



Existem duas formas de provar isso.



\### Prova construtiva



Você desenha o caminho.



```text

Entrada

&#x20;  │

&#x20;  ▼

→ → ↑ → →

&#x20;       │

&#x20;       ▼

&#x20;     Saída

```



Você mostrou exatamente como sair.



\---



\### Prova não construtiva



Você não desenha o caminho.



Apenas demonstra, por algum argumento, que \*\*é impossível o labirinto não ter uma saída\*\*.



Você sabe que ela existe, mas ainda não sabe onde está.



Essa é a principal diferença entre os dois tipos de prova:



\- A \*\*prova construtiva\*\* encontra o objeto.

\- A \*\*prova de existência não construtiva\*\* apenas garante que ele existe.
